# Tax-Mate

프리랜서의 수입·지출·세금을 쉽게 관리하고, 종합소득세 예상 금액을 미리 알려주는 서비스

---

## 🎯 타겟 유저

- 3.3% 원천징수 기반 프리랜서  
  (개발자, 디자이너, 작가, 강사, 배달 등)
- 사업자등록 없이 활동하는 프리랜서
- 종합소득세 신고가 처음이거나 매번 헷갈리는 사람

---

## 💡 핵심 가치

### 수입 한눈에
- 건별 수입 등록
- 클라이언트별 / 월별 정리

### 세금 예측
- 3.3% 원천징수 자동 계산
- 5월 종소세 예상 금액 제공

### 지출 관리
- 경비 처리 가능한 지출 분류
- 절세 도움

### 신고 준비
- 종소세 신고 시 필요한 데이터 한 번에 정리

---

## 🔐 인증 전략 (Lazy Authentication)

### 기본 전략
- 앱 진입 시 로그인 강제 ❌
- 특정 기능 사용 시 로그인 유도 ✅

### 비로그인 가능 기능
- 세금 계산기  
  (수입 금액 입력 → 원천징수 / 종소세 계산)
- 프리랜서 세금 가이드  
  (3.3% 개념, 종소세 기간 등)

### 로그인 필요 기능
- 수입 등록 / 관리
- 지출 등록 / 관리
- 월별 / 연간 리포트
- 종소세 예상 계산 (저장된 데이터 기반)
- 내 정보 관리

### 로그인 유도 UX 흐름
1. 세금 계산기 사용 (비로그인)
2. "이 기록을 저장하시겠어요?"
3. 로그인 / 회원가입 팝업
4. 저장 완료

---

# 🚀 Phase 1 - MVP

## 1.1 세금 계산기 (비로그인)

- **원천징수 계산**
  - 수입 금액 입력
  - 3.3% (소득세 3% + 지방세 0.3%) 자동 계산
  - 실수령액 계산

- **종소세 간편 계산**
  - 연 수입 + 경비 입력
  - 예상 종소세 계산 (단순경비율 적용)

---

## 1.2 수입 관리 (로그인 필요)

- 수입 등록
  - 클라이언트명
  - 금액
  - 날짜
  - 메모

- 수입 수정 / 삭제
- 수입 목록 조회
  - 월별 필터
  - 클라이언트별 필터

- 원천징수 자동 계산
  - 등록 시 3.3% 자동 차감 표시

---

## 1.3 지출 관리 (로그인 필요)

- 지출 등록
  - 항목명
  - 금액
  - 날짜
  - 카테고리
  - 메모

- 지출 수정 / 삭제
- 지출 목록 조회
  - 월별 필터
  - 카테고리별 필터

- 경비 처리 여부
  - 세금 계산 시 포함 여부 체크

---

## 1.4 대시보드 (로그인 필요)

### 월별 요약
- 총 수입
- 총 지출
- 원천징수액
- 순수익

### 연간 요약
- 연간 총 수입
- 예상 종소세
- 이미 낸 원천징수
- 추가 납부 / 환급 예상

---

## 1.5 인증

- 이메일 회원가입 / 로그인
- JWT 기반 인증
- 세금 계산기는 비로그인 사용 가능
